# `projektmanagement`

Skills rund um Planung und Anforderungen: einen Plan oder ein Design auf Herz und Nieren prüfen,
Kundenwünsche in saubere User Stories überführen, Anforderungen schärfen.

## Installation

```
/plugin install projektmanagement@3m5-skill-hub
```

## Enthaltene Skills

| Skill | Aufruf | Beschreibung |
|-------|--------|-------------|
| `grill-me` | `/projektmanagement:grill-me` | Claude stellt hartnäckige Rückfragen zu Ihrem Plan oder Design, bis jede Entscheidungsverzweigung aufgelöst ist. |
| `storytelling` | `/projektmanagement:storytelling` | Führt einen Kundenwunsch in sechs Phasen zu einer fertigen, INVEST-konformen User Story — als Jira-Issue oder Markdown. |
