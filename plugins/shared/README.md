# `shared` — für alle Abteilungen

Rollenübergreifende Skills, die in jeder Abteilung nützlich sind. Dieses Plugin sollten Sie
immer installieren.

## Installation

```
/plugin install shared@3m5-skill-hub
```

## Enthaltene Skills

| Skill | Aufruf | Beschreibung |
|-------|--------|-------------|
| `happy-wife` | `/shared:happy-wife` | Claude beantwortet Ihre Fragen als genervte, passiv-aggressive Ehefrau — inhaltlich korrekt, aber in häusliche Bitterkeit verpackt. |
