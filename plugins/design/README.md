# `design`

Skills für die Design-Arbeit: UI/UX-Feedback, Figma-Workflows und Design-Dokumentation.

## Installation

```
/plugin install design@3m5-skill-hub
```

## Enthaltene Skills

*Noch keine Skills veröffentlicht — in Entwicklung.*

Geplant sind unter anderem Skills für markenkonformes Gestalten auf Basis des 3m5. Design Systems
(Farben, Typografie, Logo-Nutzung) sowie für UI/UX-Reviews.
